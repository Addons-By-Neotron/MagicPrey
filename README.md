# Magic Prey

A World of Warcraft addon that provides a [LibDataBroker](https://www.wowace.com/projects/libdatabroker-1-1) data source for tracking Prey Hunts in The War Within.

## Features

- **LDB data source** — displays hunt state (Cold, Warm, Hot, Found) with colored text in any LDB display addon (Button Bin, ChocolateBar, Titan Panel, etc.)
- **Out-of-zone tracking** — shows hunt status and zone name even when you leave the hunt area
- **Torment display** — tooltip lists active prey modifiers/torments with descriptions
- **State-specific icons** — LDB icon changes to match the current hunt state
- **Configurable colors** — customize the color for each hunt state via the options panel
- **Hide Blizzard tracker** — optionally hide the built-in prey crystal widget from the Encounter Bar
- **Left-click** — opens the World Map centered on the prey quest
- **Right-click** — opens the addon settings in the Blizzard Interface Options

## Requirements

- An LDB display addon to show the data source (e.g., [Button Bin](https://www.curseforge.com/wow/addons/button-bin))
- World of Warcraft: The War Within (12.0+)

## Configuration

Open settings via:
- Right-click the LDB button
- Or navigate to **Interface Options > Addons > Magic Prey**

### General
- Toggle to hide the built-in Blizzard prey tracker widget

### Colors
- Color pickers for each hunt state: Cold, Warm, Hot, Found, Away

### Profiles
- Standard AceDB profile management (per-character, shared, copy, reset)
